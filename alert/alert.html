Alert

An alert displays a short, important message in way that attracts the user's attention without interrupting the user's task.

Outline Alert

Outline alerts are created by using alert class, followed by outline-variant.

Primary!This is a Primay Alert-check it now!
Error!This is an Error Alert-check it now!
Success!This is Success Alert-Check it now!
Warning!This is a warning Alert-Check it now!

Simple Alert

Simple Alerts can be created using alert class, followed by the variant name.

Primary!This is a Primay Alert-check it now!
Error!This is an Error Alert-check it now!
Success!This is Success Alert-Check it now!
Warning!This is a warning Alert-Check it now!
